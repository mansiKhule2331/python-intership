# 📝 Online Feedback Collector with Admin Dashboard

## 📌 Project Overview

The **Online Feedback Collector** is a web-based application that allows users to submit feedback through a simple interface while enabling administrators to view and analyze feedback using an admin dashboard.

This project demonstrates full-stack development using **Python Flask, HTML, CSS, JavaScript, and SQLite**.

The system collects user feedback, stores it in a database, and displays the collected responses in a structured dashboard for administrators.

---

## 🎯 Project Objectives

- Build a **web-based feedback collection system**
- Store feedback data in a **database**
- Display feedback results in an **admin dashboard**
- Understand **backend routing and database operations**
- Practice **full-stack development using Python**

---

## 🚀 Features

- 📝 User Feedback Form
- 💾 Data Storage using SQLite Database
- 📊 Admin Dashboard to view submitted feedback
- 🔄 Real-time feedback display
- 🎨 Simple and responsive interface
